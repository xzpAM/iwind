- # Iwind Frontend Assets (Static)

  This directory contains the compiled and optimized static assets for the **DeepWind** platform, the user interface for the Iwind agent.

  ## Overview

  The assets in this directory (including JavaScript, CSS, and HTML templates) are ready for production use and are automatically served by the backend when the system is launched.

  ### Accessing the Interface

  - **Primary Endpoint**: `http://localhost:8009`
  - **Asset Root**: `./static/`

  ## Proprietary Source Code Notice

  **Important: The raw frontend source code (React/Vue/TypeScript) and the original design assets are currently not included in this public distribution.**

  ### Reason for Restriction

  The Iwind platform incorporates unique UI/UX designs and high-performance visualization modules that are currently part of an **active patent application process**. To safeguard the intellectual property (IP) and technical innovations involved:

  - The source code is withheld until the patent is officially published.
  - The assets provided here are pre-compiled and obfuscated versions meant for system execution rather than development or modification.

  ## Future Availability

  We are committed to the principle of open-source research. Following the successful filing and initial publication of the patent, we intend to release the full source code under the **MIT License** to allow the community to customize and extend the Iwind interface.

  ------

  ## Technical Specifications

  - **Client-side Rendering**: Optimized for modern Chromium-based browsers.
  - **Data Communication**: Interacts with the Iwind backend via WebSocket and REST APIs.
  - **Visualization**: Supports real-time 3D structural response rendering for wind turbine models.
